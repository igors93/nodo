# Glossary

| Term | Meaning |
| --- | --- |
| Localnet | Development network profile used for local runtime tests and operator flows. |
| Testnet candidate | Official pre-testnet profile with stricter safety gates and diagnostics. |
| Mainnet | Production network profile. It is intentionally blocked in the current code. |
| Finalized artifact | Persisted record of a finalized block and its domain-specific evidence sections. |
| State root | Deterministic commitment to account state after genesis or block execution. |
| Governance vote evidence | A vote bound to a proposal envelope, voting policy, vote record, and deterministic proof. |
| Governance lifecycle | Proposal, policy, votes, tally, decision, and lifecycle metadata stored as one auditable record. |
| Treasury execution evidence | The proposal, approval, policy, treasury state, spend record, and governance context needed to verify a spend. |
| Chain audit | Local verification that reloads runtime state and checks protocol invariants. |
| Protection work | Measurable security work that can support rewards, penalties, or operational trust decisions. |
