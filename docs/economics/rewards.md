# Rewards

Nodo's reward direction is tied to Proof of Protection: rewards should reflect measurable protection work, not only passive participation.

## Current Foundations

- epoch reward block proposal records;
- epoch reward distributor tests;
- protection ledger integration;
- protection state rebuilder;
- protection economics tests;
- validator score foundations.

## Planned Hardening

Before production activation, rewards need:

- finalized protection scoring;
- clear eligibility rules;
- anti-Sybil and network risk handling;
- report-backed settlement;
- audit coverage that connects rewards to finalized evidence.
