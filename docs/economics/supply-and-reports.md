# Supply and Reports

Nodo uses reports and audit records to make economic state explainable after reload.

## Report Types

Current foundations include:

- epoch monetary reports;
- epoch treasury reports;
- supply deltas;
- supply audit results;
- finalized supply audit records.

## Audit Goal

A node should be able to answer:

- where did supply come from;
- what changed it;
- whether reports match finalized history;
- whether a reload result diverges from canonical records.

## Status

The report and audit foundations exist. Final production monetary activation is still planned work.
